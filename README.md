# Walk Finder

Interactive walking route maps generated from multiple data sources (OSM, Public Rights of Way, curated routes).

## Maps

| Location | Link |
|----------|------|
| Swanage, Dorset | [View Map](https://jtrotter83.github.io/walk-finder/swanage/) |

## Features
- Routes from multiple sources combined and deduplicated
- Colour-coded by difficulty (Easy/Moderate/Hard)
- Elevation profiles, distance, estimated time and step count
- Works on mobile browsers

## Tech
- Python script generates static HTML maps using folium
- Data from OSM Overpass API, Dorset PROW, curated walking sites
- Hosted on GitHub Pages
